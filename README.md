\# Deploy to Render



1\. Создай Web Service.

2\. Загрузи файлы проекта.

3\. В "Environment" добавь переменные:



TELEGRAM\_TOKEN = твой\_телеграм\_токен

CRYPTO\_API\_TOKEN = твой\_crypto\_api\_token



4\. Команда запуска:

python bot.py



5\. Нажми Deploy и жди 30–60 секунд.



Бот начнёт работать автоматически.
